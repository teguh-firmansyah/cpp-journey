# My C++ Development Journey

Welcome to my C++ repository! This section documents my roadmap, projects, milestone achievements, and the technical core concepts I have mastered along my journey of learning and building with C++.

---

## Roadmaps & Milestones

### 🔲 Phase 1: Foundations & Core Syntax (Fundamentals)
- [ ] **Variables & Data Types:** Understanding memory sizes, primitive types (`int`, `double`, `char`, `bool`), and modifiers.
- [ ] **Control Flow:** Mastering `if-else` statements, `switch-case`, and loops (`for`, `while`, `do-while`).
- [ ] **Functions:** Function overloading, default parameters, pass-by-value vs. pass-by-reference.
- [ ] **Arrays & Strings:** Working with C-style arrays, multidimensional arrays, and the `std::string` library.

### 🔲 Phase 2: Mastering Memory Management
- [ ] **Pointers & References:** Address-of operator (`&`), dereferencing (`*`), pointer arithmetic, and null/void pointers.
- [ ] **Dynamic Memory Allocation:** Using `new` and `delete` operators, preventing memory leaks.
- [ ] **Smart Pointers (Modern C++):** Transitioning to `std::unique_ptr`, `std::shared_ptr`, and `std::weak_ptr`.

### 🔲 Phase 3: Object-Oriented Programming (OOP)
- [ ] **Classes & Objects:** Access modifiers (`public`, `private`, `protected`), constructors, and destructors.
- [ ] **Encapsulation & Abstraction:** Hiding implementation details, creating robust APIs.
- [ ] **Inheritance & Polymorphism:** Base and derived classes, virtual functions, abstract classes, and runtime polymorphism.
- [ ] **Operator Overloading:** Customizing how operators work with user-defined classes.

### 🔲 Phase 4: Advanced C++ & Standard Template Library (STL)
- [ ] **Templates:** Writing generic code with Function Templates and Class Templates.
- [ ] **STL Containers:** Vector, List, Deque, Set, Map, and Unordered Map.
- [ ] **STL Algorithms:** Sorting, searching, modifying, and non-modifying sequences (`std::sort`, `std::find`, etc.).
- [ ] **File I/O:** Reading and writing data using `<fstream>` (`ifstream`, `ofstream`).
